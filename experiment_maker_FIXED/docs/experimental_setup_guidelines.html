⚙️ Experimental Setup Guidelines

Follow these guidelines to ensure consistent and valid experimental conditions across all participants.

Display Settings

  • Use sRGB color space if available
  • Set brightness to comfortable level (avoid glare)
  • Ensure monitor is at 60Hz or higher refresh rate

Viewing Distance

  • Position participant 50-70cm from screen
  • Eye level should align with center of screen
  • Ensure consistent positioning across participants

Environment

  • Minimize distractions and ambient noise
  • Use consistent lighting conditions
  • Experiment will run in fullscreen mode

Recommended Configuration Settings

These recommended settings provide a good balance between data quality and participant comfort for the Stroop task:

Parameter Recommended Value Notes
Number of Test Trials 18-36 trials Must be a multiple of 9 to balance conditions. 18 trials for quick screening, 36 for more robust data.
Practice Trials Enabled Recommended to ensure participants understand the task before data collection.
Number of Practice Trials 3-6 trials 3 trials minimum to familiarize participants with the task. 6 trials if participants are inexperienced.
Stimulus Delay 500ms Time between fixation and stimulus. Standard delay that prevents anticipation while maintaining engagement.
Feedback Duration 800ms Duration of feedback display. Sufficient for participants to register correct/incorrect responses.
Tip: For research purposes, we recommend using 27 or 36 test trials with 3 practice trials, 500ms stimulus delay, and 800ms feedback duration. These settings have been validated in cognitive psychology research and provide reliable Stroop effect measurements.
Note: Maintaining consistent experimental conditions is crucial for data quality and reproducibility. These guidelines apply to all cognitive experiments run on this platform.
